User Stories & Flow — 🙋 일반 사용자

키오스크를 직접 이용해 혈압·심박을 측정하는 사용자의 메인 플로우입니다.

US-U01 🙋 일반 사용자 — 얼굴 인식 측정 ver.1.0
일반 사용자로서, 출근 시 빠르게 혈압과 심박을 확인하기 위해,
키오스크 화면 앞에 서서 얼굴 인식으로 본인을 확인하고 측정을 시작한다.
US-U03 🙋 일반 사용자 — Quality Gate 탈락 재측정
일반 사용자로서, 측정 환경 문제로 부정확한 결과가 나오는 것을 방지하기 위해,
Quality Gate 탈락 안내 화면을 확인하고 자세를 바로잡아 재측정을 시도한다.
유저 플로우 — 일반 사용자 측정 플로우 (UF-01)
1
대기 화면 진입 및 동의
개인정보 및 민감정보 수집 이용 동의 선택
2
본인 인증 (얼굴/이름)
카메라 인증 시도 → 실패 시 수동 입력
3
안정화 및 Quality Gate 체크
30초 대기 중 조도·움직임·화질 자동 판정
4
rPPG 측정 및 결과 표시
AI 모델 수치 추정 → 정상/주의/위험 판정 색상 노출
5
데이터 저장 및 종료
비식별 암호화 저장 후 대기 화면 복귀
User Stories & Flow — 🔐 현장 관리자

납품처 현장에서 운영을 담당하는 관리자의 스토리입니다.

US-A01 🔐 현장 관리자 — 관리자 페이지 설정 ver.1.0
현장 관리자로서, 현장 기준에 맞는 혈압 허용 범위를 설정하기 위해,
관리자 페이지에서 헬스케어 범위와 화면 설정을 직접 관리한다.
US-A03 🔐 현장 관리자 — 커프 2차 검증
현장 관리자로서, AI 측정값이 불확실할 때 더 정확한 판단을 내리기 위해,
커프 혈압계로 2차 검증을 수행하고 최종 출입 여부를 결정한다.
