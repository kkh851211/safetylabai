📅 2026.03.04 추가 확정 — 원격 앱 관리 (기기 원격 리셋) · 지점별 통계 관리
🔐 숨겨진 관리자 페이지
▼
  • 수축기/이완기 헬스케어 범위 설정, 화면 밝기, 카메라 선명도 조절은 패스워드 잠금 숨겨진 버튼을 통해서만 접근 가능
  • 일반 사용자가 임의로 설정을 변경하지 못하도록 관리자 기능을 완전히 분리
  • 사용자 화면의 얼굴 선명도는 결과에 영향을 미치지 않는 선에서 낮게 유지
  • 기능 전체 목록을 개발 착수 전 양사 합의로 최종 확정 필요
🛡️ 원격 관리 및 보안 시스템
▼
  • 원격 유지보수: 코레일, SRT 등 외부 납품 시 인터넷 기반 원격 관리 기능 필요
  • 라이선스 인증키: 타 업체의 무단 복제·열람 방지를 위해 하드웨어 고유 번호 기반 인증키 발급 시스템 구축
  • 개인정보 보호: 이름 등 개인정보는 암호화하여 비식별 정보로 DB 저장, 클라우드(AWS 등) 연동 시 유출 엄격 방지
🪪 인증 방식 다각화
▼
  • 기본: 얼굴 인식
  • 추가: 키보드로 이름 직접 입력 (얼굴 인식 불편 사용자 대응)
  • 확장 예정: RFID 카드 리더기, 지문 인식기 등 출입 통제 디바이스 연동
🤖 AI 업데이트 방식 확정
▼
  • 실시간 업데이트 방식 불채택
  • 데이터를 충분히 축적한 뒤 3개월 단위 버전 업데이트 방식으로 고도화
  • 각 버전 업데이트 시 캘리브레이션 재수행 및 정확도 검증 포함
🔍 Quality Gate (측정 품질 자동 판정)
▼
  • 측정 환경(조도·움직임·카메라 품질) 자동 판정, 기준값 수치 정의
  • Quality Gate 탈락 시 재측정 안내 화면 표시
  • 부재 시 오탐 발생 → 최대 리스크, 초기 개발에서 반드시 구현
📱 원격 앱 관리 (기기 원격 리셋) 03.04 추가
▼
  • 원격 리셋·재시작: 관리자 앱/웹에서 특정 키오스크를 선택하여 기기 강제 리셋 또는 재시작 명령 전송
  • 원격 SW 배포: 소프트웨어 업데이트 패키지를 원격으로 배포하고 설치 진행 상태 확인
  • 상태 모니터링: 기기별 온라인/오프라인 상태, 마지막 통신 시각, 오류 로그 실시간 확인
  • 이상 알림: 장시간 오프라인, 반복 오류 등 감지 시 관리자에게 푸시 또는 이메일 알림 자동 발송
  • 접근 보안: 원격 제어 기능은 인증된 관리자 계정에서만 실행 가능하도록 권한 분리 설계 필요
  • ⚠ 보안 권장: 리셋·SW 배포 등 고위험 명령은 2단계 인증(MFA/OTP) 적용 권장, 명령 이력 로그 의무 보관
📊 지점별 통계 관리 03.04 추가
▼
  • 지점별 집계: 납품처(가온병원, 코레일 등)를 지점 단위로 구분하여 측정 건수, 혈압 분포, 출입 현황 집계
  • 추이 분석: 일별·주별·월별 혈압 측정 추이를 차트로 시각화하고 지점 간 비교 기능 제공
  • 출입 현황 리포트: 출입 허가·거부 건수, 측정 불가(Quality Gate 탈락) 비율 등 운영 지표 제공
  • 데이터 내보내기: 지점별·기간별 통계 데이터를 CSV 또는 Excel 형식으로 내보내기 지원
  • 개인정보 처리: 통계 화면에는 비식별·집계 데이터만 노출, 개인 측정 원본에 대한 직접 접근 차단
🏥 향후 확장 계획 (Total Healthcare)
▼
  • 단기: 음주 측정기, 폐활량 측정기 연동 → 생활 의료기기 수준의 토탈 헬스케어 기기로 발전
  • 중장기: 정부 과제(보건복지부 등) 연계, 모바일 앱 및 스마트폰 카메라 기반 생체 신호 측정 시스템 확장
  • 활용 시나리오: 요양보호사 방문 시 스마트폰으로 건강 기록 측정 후 서버 전송

AI Enhancement Plan

커프 혈압계를 정답지로 활용한 AI 모델 학습 및 고도화 계획입니다.

학습 방식
  • 정답지 생성: 커프 혈압계로 측정한 실제 혈압값을 AI 학습의 정답 데이터로 활용
  • 입력 데이터: 카메라로 수집한 심박 및 파형 데이터
  • 교차 검증: AI 추정값과 커프 측정값을 동시에 비교·검증하며 모델 보정
  • 업데이트 방식: 3개월 단위 버전 업데이트 (실시간 업데이트 불채택)
데이터 수집 계획
  • 최소 필요: 50~100명 이상 피험자 데이터
  • 1차 수집: 코레일 종사자 약 170명 대상 생체 데이터 수집
  • 2차 수집: 학술대회, 코엑스 박람회 등 키오스크 배치로 불특정 다수 데이터 확보
  • 지속 수집: 운영 현장(가온병원 등)에서 누적 데이터를 3개월마다 모델에 반영
  • 지점별 품질 관리: 지점별 통계 기능으로 납품처별 데이터 수집 현황을 모니터링하여 AI 학습용 데이터 품질 관리 03.04 추가
