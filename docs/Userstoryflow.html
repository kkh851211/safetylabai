User Stories & Flow — 🙋 일반 사용자

키오스크를 직접 이용해 혈압·심박을 측정하는 직원 또는 방문객의 유저 스토리와 메인 측정 플로우입니다.

US-U01 🙋 일반 사용자 — 얼굴 인식 측정 핵심
일반 사용자로서, 출근 시 빠르게 혈압과 심박을 확인하기 위해,
키오스크 화면 앞에 서서 얼굴 인식으로 본인을 확인하고 측정을 시작한다.
  • 수용 조건: 얼굴 인식 성공 시 3초 이내 안정화 유도 화면으로 전환
  • 수용 조건: 측정 중 움직임 감지 시 "잠시 정지해 주세요" 안내 표시
  • 수용 조건: 측정 완료 후 혈압·심박 수치 및 헬스케어 범위 판정 결과 표시
US-U02 🙋 일반 사용자 — 이름 입력 대체 인증 핵심
일반 사용자로서, 마스크 착용 등으로 얼굴 인식이 어려울 때 측정을 포기하지 않기 위해,
키보드로 이름을 직접 입력하여 인증을 완료하고 측정을 진행한다.
  • 수용 조건: 얼굴 인식 실패 시 "이름으로 입력하기" 버튼 화면에 표시
  • 수용 조건: 이름 입력 후 동일한 측정 플로우 진행
US-U03 🙋 일반 사용자 — Quality Gate 탈락 재측정 핵심
일반 사용자로서, 측정 환경 문제로 부정확한 결과가 나오는 것을 방지하기 위해,
Quality Gate 탈락 안내 화면을 확인하고 자세를 바로잡아 재측정을 시도한다.
  • 수용 조건: 탈락 이유(조도 부족 / 움직임 감지 / 화질 불량)를 구체적으로 안내
  • 수용 조건: 재측정 버튼 1회 탭으로 즉시 재시도 가능
  • 수용 조건: 연속 3회 탈락 시 커프 혈압계 안내 화면으로 전환
US-U04 🙋 일반 사용자 — 결과 확인
일반 사용자로서, 오늘 측정한 혈압·심박 결과가 정상 범위인지 확인하기 위해,
측정 결과 화면에서 수치와 판정 색상(정상/주의/위험)을 확인한다.
  • 수용 조건: "본 측정값은 의료 진단이 아닙니다" 면책 문구 항상 표시 (법적 필수)
  • 수용 조건: 정상/주의/위험 범위는 관리자가 설정한 헬스케어 기준값 기반으로 표시
US-U05 🙋 일반 사용자 — 개인정보 동의 및 저장
일반 사용자로서, 내 건강 데이터가 어떻게 사용되는지 알고 동의하기 위해,
개인정보 수집·이용 동의 화면을 확인하고 측정 전에 동의 여부를 선택한다.
  • 수용 조건: 얼굴 인식 별도 동의, 민감정보(혈압) 별도 동의 항목 구분 표시
  • 수용 조건: 미동의 선택 시 측정 불가 안내 후 종료
  • 수용 조건: 동의 내역은 암호화하여 비식별 처리 후 저장

User Stories & Flow — 🔐 현장 관리자

납품처(병원·철도사)에서 키오스크를 직접 운영·설정하며 출입 판단을 보조하는 현장 관리자의 유저 스토리와 플로우입니다.

US-A01 🔐 현장 관리자 — 관리자 페이지 설정 핵심
현장 관리자로서, 현장 기준에 맞는 혈압 허용 범위를 설정하기 위해,
숨겨진 패스워드 버튼으로 관리자 페이지에 접근하여 수축기/이완기 헬스케어 범위와 화면 밝기를 설정한다.
  • 수용 조건: 패스워드 입력 오류 5회 시 일정 시간 잠금 처리
  • 수용 조건: 설정 변경 내역은 관리자 로그에 기록 (변경 시각·항목 포함)
  • 수용 조건: 일반 사용자 화면에서는 관리자 버튼이 보이지 않음
US-A02 🔐 현장 관리자 — 출입 판단 보조 핵심
현장 관리자로서, AI 측정 결과를 바탕으로 안전한 출입 여부를 판단하기 위해,
관리자 화면에서 측정 결과와 판정 색상을 확인한 뒤 출입 허가 또는 거부를 최종 결정한다.
  • 수용 조건: AI는 출입 판단을 보조하는 역할에 한정 — 최종 판단은 반드시 관리자가 수행
  • 수용 조건: 허가/거부 결정 내역은 타임스탬프와 함께 로그 저장
US-A03 🔐 현장 관리자 — 커프 2차 검증 핵심
현장 관리자로서, AI 측정값이 불확실하거나 주의 범위일 때 더 정확한 판단을 내리기 위해,
커프 혈압계로 2차 검증을 수행하고 최종 출입 여부를 결정한다.
  • 수용 조건: 커프 2차 검증 루트는 항시 제공 — AI 단독 판단 구조 불가
  • 수용 조건: 커프 측정값은 AI 추정값과 함께 캘리브레이션 데이터로 자동 저장
US-A04 🔐 현장 관리자 — 캘리브레이션 실험 모드
현장 관리자로서, AI 모델 고도화를 위한 정답 데이터를 수집하기 위해,
실험 모드를 활성화하여 AI 추정값과 커프 측정값을 동시에 기록하고 세션을 저장한다.
  • 수용 조건: 실험 모드는 관리자 페이지에서만 활성화 가능
  • 수용 조건: 세션 단위로 AI 추정값·커프 측정값·라벨링 항목 일괄 저장
유저 플로우 — 현장 관리자 출입 판단 플로우 (UF-02)

✓ 허가 → 출입 게이트 연동 신호 ✗ 거부 → 사용자에게 거부 안내
